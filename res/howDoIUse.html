How do I use this program?

This program uses PlantUML language, for creating UML diagrams right from your java source code (only Java 7 and below, is supported). Output is PlantUML code, which can be interpretated by PlantUML software or plugins for different IDE's. Check out the PlantUML web-site to get more information, about how you can turn PlantUML code, into the diagram. Nevertheless, if you don't want to perform additional actions, then our program and installed Graphviz software, is the only things that you need, to see diagram of your Java project. If your java source code is too big and your computer is slow, we're recommending to you to turn off the option "Show diagram", and generate only PlantUML code. After that you could click the button "Open diagram in browser". That will generate diagram using PlantUML server and will open it on PlantUML website. Below you will find instructions on how to create a chart using only this program.

  • In first of all, if you want to see image of diagram in "Diagram" tab, you should install Graphviz software. It can be downloaded from Graphviz web-site. Installation process for different operating systems may vary.

  • Then you should choose directory of your Java project, or jar archive, by clicking the button "Select files".

  • Then, optionally, set preferred parametres of future diagram. Such as direction of diagram, shown relations, type of files to be parsed, and preferred type of diagram. Some parameters that you'll see, may be unavailable to use, that simply means, that we doesn't implement that functionality yet. Watch for changes.

  • Press "Generate" button.

  • If all previous steps has been done correct, in tab "PlantUML code" you will see PlantUML code, and in "Diagram" tab, you'll see UML diagram that represents your project.

If Graphviz installed incorrect in your system, and you're trying to generate classes diagram, you will see error message, generated by PlantUML, in the "Diagram" tab.
